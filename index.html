About The Calendar

所有代码均使用原生 JavaScript 手写，产品文档使用 JSDoc 生成。 刚接触前端不久，还有很多地方需要提高，之后会继续进行学习，完善这个日历。

在这个任务中，日历生成算法主要受到The Strange Zen Of JavaScript里这篇文章的启发，CSS的知识掌握主要在学习CSS布局这个网站中得到提高，JavaScript则是通过Coursera上的HTML, CSS and JavaScript课程进行了学习。

同时翻阅的书籍有：

  • HTML & CSS
  • JavaScript & jQuery交互式Web前端开发
  • JavaScript DOM编程艺术
  • CSS权威指南
